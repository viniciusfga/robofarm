# RoboFarm Idle - Vision

## 🎯 Proposta
Jogo incremental onde o jogador automatiza uma fazenda com robôs.

## 🧠 Core Loop
Produzir → Vender → Melhorar → Automatizar

## 🎮 Diferencial
- Foco em automação inteligente
- Evolução para sistema semi-autônomo
- Possível integração com IA futuramente

## 👤 Público
- Jogadores casuais
- Interessados em automação/simulação
